Add a widget for Awasete Yomitai.

== Description ==

This plugin add a widget for Awasete Yomitai. Awasete Yomitai is Japanese service and display blog list realated your blog.
`Awasete Yomitai` means "Want to read with"(?)

== Installation ==

1. Upload `awasete-yomitai-for-wordpress` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Widget  through the 'Appearance' > 'Widget' menu in WordPress

== Frequently Asked Questions ==

= What about Awasete Yomitai? =

Awasete Yomitai (http://awasete.com/) is a web service to display related blog in Japan.
